This project solves and simulates for Airbnb model.

Here all a brief description of all codes
main: the main code to run

simulation: simulate for moments

residuals: calculate residuals for equations

cheb:calculate Chebyshev polynomials

cartesianproduct5: calculate cartesian product of dimension 5

chebyextrema: calculate the Chebyshev extremas

hernodes: calculate Gauss-Hermite nodes

itransfo: transform [-1,1] to [min,max]

transfo: transform [min,max] to [-1,1]
